# allowed_posters.txt  (supplemental — usually not needed)
#
# By default, blog posts are published from any open GitHub Issue authored by
# the repository owner or any collaborator with write (push/maintain/admin)
# access.  You do not need to list those users here.
#
# Use this file only if you want to grant posting rights to a GitHub user who
# does NOT have repository write access (e.g. a trusted external contributor).
#
# Rules:
#   - Add one GitHub username per line.
#   - Lines starting with # are comments and are ignored.
#   - Add a single * on its own line to allow ALL users to create blog posts
#     (open-contributor mode — use with caution).
#
# Examples:
#   alice
#   bob
#   *
