# allowed_posters.txt
#
# Controls which GitHub users can author blog posts (i.e. whose Issues with the
# "blog-post" label will be published on the site).
#
# Rules:
#   - The repository owner is ALWAYS allowed regardless of this file.
#   - Add one GitHub username per line to grant additional users posting rights.
#   - Lines starting with # are comments and are ignored.
#   - Add a single * on its own line to allow ALL users to create blog posts
#     (open-contributor mode — use with caution).
#
# Examples:
#   alice
#   bob
#   *
