First blog post

By HanClinto — April 13, 2026
blog-post

So I've wanted to have a new blog for a long time, but I haven't fully settled on architecture until now. The idea with this blog is to:

1) Be easily hostable on Github Pages
2) Be easily copyable for others who may want to host their own blog -- even if they aren't very git-savvy.
3) Utilize Github Pages (normally for static sites only) in a way that feels dynamic. Do this by leveraging features of Github itself to translate dynamic project elements (issues, comments, pipelines) into blog posts, comments, etc.
4) Promote social interaction by encouraging comments, and cross-linking to other users who have their own blogs.

So this is an experiment along those regards. We'll see how well it works? Feel free to comment here if you want to be part of the experiment!
